· assembler

how this works

Each strip is a list of numbers. Numbers come in threes — red, green, blue — for every pixel in that strip.

Strip 1 holds the top 32 rows of the image. Strip 2 the next 32. Then 3, then 4. Each strip is 4,096 pixels = 12,288 numbers.

Paste each strip into the matching box. Press assemble. The page reads the numbers and paints them, three at a time, onto the canvas. R, G, B, next pixel. R, G, B, next pixel. Until all 16,384 pixels are drawn.

If a strip is empty or has the wrong count, you get an error and no image. Order matters: strip 1 paints first (top of the image), strip 4 paints last (bottom).
